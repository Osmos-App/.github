# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues by emailing: **security@useosmos.com**

Include:
- Type of vulnerability
- Affected component (osmos-core, osmos-macos, etc.)
- Steps to reproduce
- Potential impact

## Response

- You will receive acknowledgement within 48 hours
- We will keep you informed throughout the remediation process
- Confirmed vulnerabilities will be credited to the reporter (unless anonymity is requested)

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |
| Older   | ❌ |
